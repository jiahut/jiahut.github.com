Use Git Impove Process From Coding to Testing

Mar 19th, 2013

使用git改善代码编写与测试的过程

  1. 挂载待测试代码目录，这要得益于git原生支持共享文件系统
    • win
      • net share project_1=e:\projects\rails /grant:administrator,full
        • net share project_1 /del
      • net use z: \\<HOST>\project_1
        • net use z: /del
    • linux
      • mount
      • 使用ssh是更好的方案

  2. fork一份拷贝,checkout到bugXXX分支作修正

    • git clone z: project_1
    • git branch bugXXX
    • git checkout bugXXX

  3. push你的修正

    • git config --add receive.denyCurrentBranch ware

      • cd z: && git config --add receive.denyCurrentBranch ware

      • or git config -e or vim .git\config append bellow config

          [receive]  
  denyCurrentBranch = ware

    • git checkout master

    • git merge or rebase bugXXX
    • git push
